
# Система прогнозного мониторинга и авто-восстановления

Дипломный проект по специальности «Сетевое и системное администрирование».

## Описание

Система автоматически собирает метрики веб-приложения, прогнозирует рост нагрузки и заранее запускает дополнительные контейнеры.
